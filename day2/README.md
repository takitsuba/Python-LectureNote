# 機械学習 x Python
ここでは、線形回帰アルゴリズムを題材に、機械学習の基礎を学びつつ、それをPythonで実装します。

## 前提パッケージ
以下2つのパッケージを利用しますので、`pip`でインストールしてください。
```
$ pip install numpy
$ pip install matplotlib
```

## 演習内容
`ex1.py`と`ex2.py`について、穴埋めと成っている部分を実装してください。  
具体的には、それぞれのファイルで以下の内容を実装します。
 - 仮説関数
 - コスト関数
 - 最急降下法

## 演習の答え
